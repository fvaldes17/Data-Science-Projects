 Módulos:

 1.  Análisis Base de Datos EMNIST.ipynb: Este módulo contiene una muestra de como es la base de datos EMNIST.
 2.  Análisis de la Base de Datos Inicial.ipynb: Este módulo implementa la carga de la base de datos Digits y sus métodos de reducción de dimensionalidad.
 3.  Análisis de la Base de Datos MNIST.ipynb: Este módulo contiene una muestra de como es la base de datos MNIST y sus métodos de reducción de dimensionalidad.
 4.  Conversor DATABASE Propia.ipynb: Este módulo contiene una muestra de como son las bases de datos creadas por los integrantes del grupo 1.
 5.  database_creada.py: Este módulo implementa la carga de las bases de datos creadas por los integrantes del grupo 1.
 6.  DecisionTree EMNIST.ipynb: Algoritmo de clasificación Decision Tree aplicado en EMNIST desbalanceado y balaceado, además de la clasificación de las bases de datos creadas por los integrantes del grupo 1.
 7.  DecisionTree.ipynb: Algoritmo de clasificación Decision Tree aplicado en MNIST.
 8.  EMNIST.py: Este módulo implementa la carga de la base de datos EMNIST utilizando Tensorflow.
 9.  GaussianNaiveBayes EMNIST.ipynb: Algoritmo de clasificación Gaussian Naive Bayes aplicado en EMNIST desbalanceado y balaceado, además de la clasificación de las bases de datos creadas por los integrantes del grupo 1.
 10. GaussianNaiveBayes.ipynb: Algoritmo de clasificación Gaussian Naive Bayes aplicado en MNIST.
 11. LogisticRegression EMNIST.ipynb: Algoritmo de clasificación Logistic regression aplicado en EMNIST desbalanceado y balaceado, además de la clasificación de las bases de datos creadas por los integrantes del grupo 1.
 12. LogisticRegression.ipynb: Algoritmo de clasificación Logistic Regression aplicado en MNIST.
 13. MLPC EMNIST.ipynb: Algoritmo de clasificación Multilayer Perceptron aplicado en EMNIST desbalanceado y balaceado, además de la clasificación de las bases de datos creadas por los integrantes del grupo 1.
 14. MLPC.ipynb: Algoritmo de clasificación Multilayer Perceptron aplicado en MNIST.
 15. MNIST.py: Este módulo implementa la carga de la base de datos MNIST utilizando Tensorflow.
 16. RandomForest EMNIST.ipynb: Algoritmo de clasificación Random Forest aplicado en EMNIST desbalanceado y balaceado, además de la clasificación de las bases de datos creadas por los integrantes del grupo 1.
 17. Random Forest.ipynb: Algoritmo de clasificación Random Forest aplicado en MNIST.
